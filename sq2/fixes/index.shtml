Space Quest 2 Fixes

 

Problems running SQ1 on Windows 2000, ME, NT and XP?

If you're having problems running Space Quest 2 on a modern computer, try DOSBox. It's a program that emulates the DOS environment.

 

All the messages and the menu are unreadable

If all the texts of the game come out garbled (I.E in scrambled blocks rather than text), try running the game with the -s -v parameters. In DOS Prompt, run the game as follows: "sierra.com -s -v". Otherwise, use the NAGI program.
